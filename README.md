# Summative-1-Portfolio-of-Evidence

Sweet Haven Bakery Website

Overview

This project is a simple, fully responsive website for a fictional small business called Sweet Haven Bakery. The site was designed to reflect the warm, homemade feeling of a real local bakery and features clean navigation, pastel colors, and useful sections such as a product menu, booking form, and contact information.

The project includes five HTML pages styled with a shared CSS file. It is intended as a prototype for a small business or local shop looking to build an online presence.

Style and Design

Colors: Soft pastels (cream, peach, chocolate brown) to reflect a cozy, inviting bakery theme.
Typography: Clean sans-serif font for readability and elegance.
Layout: Mobile-friendly design using CSS media queries and responsive sizing.
Accessibility: Includes alt text, labels for form inputs, and good contrast for readability.
How to Use

Make sure all files (.html and style.css) are in the same folder.
Open index.html in a browser to view the homepage.
Navigate through other pages via the menu links.
To customize content, open the HTML files in a code editor.
Educational Purpose

This website was created as part of a coursework project to demonstrate basic front-end web development skills. It serves as an example of how HTML and CSS can be used to build a user-friendly website for a small business.
